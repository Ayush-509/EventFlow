# 🌟 Overview

**EventFlow** is a comprehensive event management platform built using the MERN stack. It enables users to discover and book events, organizers to create and manage events, and administrators to oversee platform activities.

The platform offers secure authentication, ticket generation with QR codes and barcodes, online payments, analytics dashboards, attendee management, real-time communication, reviews, favorites, and location-based event exploration.

Whether it's a workshop, concert, seminar, meetup, festival, or conference, EventFlow provides a complete end-to-end solution for managing events efficiently.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/Ayush-509/EventFlow.git
```

```bash
cd EventFlow
```

---

## Backend Setup

```bash
cd backend
npm install
npm run dev
```

---

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

# 🔑 Environment Variables

Create a `.env` file inside the backend directory.

```env
PORT=

MONGO_URI=

JWT_SECRET=

GOOGLE_CLIENT_ID=

GOOGLE_CLIENT_SECRET=

EMAIL_USER=

EMAIL_PASS=

RAZORPAY_KEY_ID=

RAZORPAY_KEY_SECRET=

CLOUDINARY_CLOUD_NAME=

CLOUDINARY_API_KEY=

CLOUDINARY_API_SECRET=
```
